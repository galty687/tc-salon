第九次技术传播沙龙

内容管理与信息分类

2021年7月31日 14:00-18:00

腾讯会议 ID：413 541 585

技术写作大体包括四个环节：一是内容创作，主要涉及以用户为中心的内容设计和使用各种技术创作内容；二是内容管理，有了海量内容后，如何对内容进行有效组织从而能高效使用内容；三是内容交付，新时代的用户获取信息的媒介在发生翻天覆地的变化，如何向用户提供及时可用的内容是一个挑战；四是内容追踪，内容交付给用户后，追踪用户如何使用这些信息，从而能持续改进内容。


本次沙龙活动聚焦内容管理与信息分类在技术写作行业中的应用。

TC Seminar 9

Content management and information classification
14:00-18:00
July 31, 2021 

Technical writing generally includes four parts. The first is content creation, which mainly involves user-centered content design and the use of various technologies to create content. The second is content management, which means how to effectively organize a large amount of content so that it could be used efficiently. The third is content delivery. In the new era, the media for users to obtain information has completely changed. It has become a challenge to provide users with timely and available content. The fourth is content tracking. After the content is delivered to users, track how users apply these information so that the content can be continuously improved.


This event will focus on the application of content management and information classification in the technical writing industry.
