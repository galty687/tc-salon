DITA 培训（中级）

Darwin Information Typing Architecture

什么是 DITA？

DITA (Darwin Information Typing Architecture) 是OASIS技术标准。DITA利用XML技术框架，构建了一套完整的用于文档的“设计”、“写作”、“管理”和“发布”的方案。对于印刷内容和在线内容均能适用。

为什么选择 DITA？

支持模块化写作

DITA定义了多种标准的信息模块，即 Concept，Task，Reference，Troubleshooting 这四种 Topic，同时DITA也支持定义新模块(Topic)。标准化的模块(topic)可便于分工协作写作，也便于信息的复用。

支持同源开发

DITA使得相同的信息只需写作一次，便可应用在各类终端的各种格式中。例如一个Topic的信息，可以同时用于“网站”、“联机帮助”，“PDF手册”等不同输出格式。避免信息冗余，提高了信息开发效率，同时也降低了开发成本。

内容与格式分离

得益于XML技术，基于DITA的写作可以实现内容与格式的分离，写作团队可专注于内容创作，无需操心呈现格式。格式工程师可利用XSLT技术将DITA信息转换为各种格式。


开源免费

DITA是一项标准，并非产品，因此无需缴纳授权费用。对于预算有限的团队，甚至可以基于开源免费工具实现低成本部署，如使用Eclipse，Visual Studio等编辑器进行写作，使用SVN进行版本控制，使用WebDav管理内容，使用DITA-OT发布文档。预算充足的团队，也可使用定制的商业工具，从而更为高效地利用该技术。
